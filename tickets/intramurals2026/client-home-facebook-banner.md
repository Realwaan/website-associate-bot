# Add Daily Facebook Updates Banner to Home Page

## Problem

The home page hero/landing section should include a prominent banner promoting daily updates from the official Facebook page. Currently, no Facebook integration or banner exists.

## Potentially Related Files

- [components/public/home/hero-section.tsx](../app/components/public/home/hero-section.tsx) — Main hero section of home page
- [components/public/home/section-wrapper.tsx](../app/components/public/home/section-wrapper.tsx) — Layout component for home sections
- [app/(public)/page.tsx](../app/app/(public)/page.tsx) — Home page entry point

## What to Fix

1. Create or add Facebook banner section in home page
2. Include a call-to-action button linking to the official Facebook page
3. Style banner to match existing branding (crimson/primary color scheme)
4. Add icon and copy about daily updates
5. Position banner prominently (near top or hero section)
6. Make mobile-responsive

## Acceptance Criteria

- Facebook banner is visible on home page desktop and mobile
- Banner includes link to Facebook page (external)
- Copy mentions daily updates 
- Design matches existing aesthetic (colors, typography)
- Button is clickable and opens Facebook in new tab
- Banner has call-to-action copy
